# clanker
Deploy and manage Clanker ERC-20 tokens on Base and Arbitrum — launch tokens, search by creator, and claim LP fee rewards.

## Highlights
- Deploy ERC-20 tokens directly on Base via Clanker V4 factory
- Search tokens by creator address or Farcaster username
- List recently deployed tokens with pagination
- Get real-time token info and price data
- Claim LP fee rewards for deployed tokens
- Built-in security scanning for token safety
- MEV protection and automated liquidity setup
- Cross-chain support (Base and Arbitrum One)
