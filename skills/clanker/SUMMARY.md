# clanker
Deploy and manage Clanker ERC-20 tokens on Base and Arbitrum — launch tokens, search by creator, and claim LP fee rewards.

## Highlights
- Deploy ERC-20 tokens directly on Base via Clanker V4 factory with no API key required
- Search tokens by creator address or Farcaster username
- List recently deployed tokens with pagination and sorting options
- Get real-time token metadata and price information from on-chain sources
- Claim LP fee rewards for deployed tokens with automatic security scanning
- Built-in MEV protection and fee structures for token deployments
- Support for Base and Arbitrum One networks
- Automatic security scanning to prevent interaction with malicious tokens
