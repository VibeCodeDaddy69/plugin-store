# hyperliquid
Trade perpetuals on Hyperliquid with full on-chain settlement, supporting market/limit orders, TP/SL brackets, position management, and USDC deposits.

## Highlights
- Check open perpetual positions and unrealized PnL on Hyperliquid L1
- Get real-time market prices for all perpetual trading pairs
- Place market/limit orders with optional stop-loss and take-profit brackets
- Close positions with one command using automatic size detection
- Set stop-loss and take-profit on existing positions
- Cancel open orders by order ID
- Deposit USDC from Arbitrum to Hyperliquid via official bridge
- Register signing address for seamless onchainos wallet integration
