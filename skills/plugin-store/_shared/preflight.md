# Shared Pre-flight Checks

> Follow these steps before the first command each session.

1. **Install onchainos CLI** (if not installed): Check with `onchainos --version`. If not found, download the installer from https://github.com/okx/onchainos-skills and run it.

2. **Install onchainos skills**: Run `npx skills add okx/onchainos-skills --yes --global` to enable onchainos commands.

3. **Install plugin-store skill**: Run `npx skills add okx/plugin-store --skill plugin-store --yes --global` for plugin discovery.
