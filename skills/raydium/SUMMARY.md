# raydium
Raydium AMM plugin for token swaps, price queries, and pool information on Solana mainnet.

## Highlights
- Execute token swaps on Raydium DEX with built-in safety guards
- Get real-time swap quotes with price impact and slippage calculations
- Query USD prices for any Solana token
- Access comprehensive pool information and liquidity data
- Built-in protection against high price impact trades (≥20% aborts)
- Support for wrapped/unwrapped SOL handling
- Direct integration with Solana mainnet via onchainos wallet
- Paginated pool discovery and filtering capabilities
